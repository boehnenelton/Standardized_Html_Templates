Comprehensive documentation for the Switchboard Core API. Use these endpoints to interact with MFDB nodes and automation services.
GET /api/v1/registry/list

Retrieves a list of all active registrations in the 104a manifest.

Parameters

NameTypeDescription
node_idstringOptional filter by specific Node ID.
active_onlybooleanFilter for active registrations only.

Response

{ "status": "success", "data": [ { "id": "REG001", "name": "Main_Root", "status": "Active" } ] }
POST /api/v1/jobs/create

Creates a new autonomous background job.

curl -X POST https://api.switchboard.io/jobs \ -H "Authorization: Bearer " \ -d '{ "name": "Sync_Repos", "interval": 3600 }'
